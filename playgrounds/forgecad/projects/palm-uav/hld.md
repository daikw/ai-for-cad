# Palm UAV — High-Level Design

手のひらサイズ（対角ホイールベース 75 mm 級、上限 100 mm）・全備重量 100 g 未満のクアッドコプター型 UAV。
屋内点検・教育・飛行制御実験・短距離 FPV を想定した 1S 電源のマイクロ機。

初版ドラフトは ChatGPT (Pro) との構想設計セッション（2026-07-06）を元に、
このリポジトリの HLD 形式へ再構成したもの。

## Problem

### 要求

| 項目 | 要求 |
|---|---|
| 機体形式 | X 型クアッドコプター（4 ローター対角配置） |
| 対角ホイールベース | 75 mm 目標、最大 100 mm 以下 |
| 全備重量 | 35〜45 g 目標、上限 100 g 未満（バッテリー・ペイロード込み） |
| プロペラ径 | 35〜40 mm 級 |
| 電源 | 1S LiHV/LiPo 300〜450 mAh |
| 飛行時間 | 3.5〜5 分目標 |
| ペイロード | 標準 3〜6 g、短時間試験なら〜10 g |
| 用途 | 屋内点検、教育、飛行制御実験、短距離 FPV、軽量センシング。屋外は微風以下に限定 |
| 操縦・制御 | 手動安定化（Angle/Acro）。高度維持・光学フローは後続検討。GNSS はこのサイズでは優先度低 |
| 安全 | プロペラガード、フェイルセーフ、低電圧保護、アームロック。屋内運用前提で安全側に振る |

### 要求の優先順位

このサイズではペイロードや自律センサを載せるほど飛行時間と操縦性が急激に悪化する。
初期設計は次の優先順で判断する。

1. 全備重量と重心管理
2. 安定して浮く推力余裕（実用推力重量比 2.0 以上）
3. フレーム剛性とプロペラクリアランス
4. 飛行時間
5. ペイロード
6. 自律機能・映像機能

まず「バッテリー込み 35〜40 g 級で確実に飛ぶ」構成を成立させ、その後にペイロードを追加する。

### 法規制約

- 日本ではバッテリー込み 100 g 未満のマルチローターは「模型航空機」扱い（航空法の機体登録対象外）。**100 g 未満は設計上のハード制約**
- ただし小型無人機等飛行禁止法は 100 g 未満も対象になり得る。飛行場所の確認は運用側の責務として別途必要
- 送信機・受信機・VTX は電波法（技適）確認必須。ELRS 機器は技適マークの有無を確認して選定する

### このリポジトリでのスコープ

ForgeCAD でモデリングするのは機体の構造部分：フレーム（アーム・中央プレート・プロペラガード/ダクト）、
FC マウント、バッテリーホルダ、センサ/カメラマウント。モーター・FC・バッテリー等の電装は市販品を
採寸してインターフェース設計のみ行う。フライトファームウェア（Betaflight）設定は CAD スコープ外。

## Approach

**1S Whoop 型マイクロクアッド**。中央に FC/ESC/RX 一体の AIO 基板、四隅に 0802 ブラシレスモーター、
中央下部に 1S バッテリー、上部にカメラ/センサを載せる。フレームは 3D プリント（PA12/ナイロン系）で
薄肉プロペラガード付きとする。

```
上面視                          側面視
   M1 CW          M2 CCW
    ○---------------○           [カメラ/センサ]
     \             /             ┌──────────┐
      \  ┌─────┐  /              │ AIO FC   │← 26×26mm マウント
       \ │ AIO │ /               ├──────────┤
       / │ FC  │ \               │ バッテリー │← 下部スロット
      /  └─────┘  \              └──────────┘
     /             \            ═╧══════════╧═ ガード下端
    ○---------------○            ↑ 床クリアランス
   M4 CCW          M3 CW
```

### サブシステム構成

| サブシステム | 採用 | 備考 |
|---|---|---|
| モーター | 0802 サイズ、19000〜19500 KV、1S ×4 | 約 1.8 g/個。BETAFPV 0802SE / Happymodel EX0802 級 |
| プロペラ | 40 mm 3 ブレード、1 mm 軸 | 約 0.42 g/枚。飛行時間試験用に 2 ブレードも比較 |
| FC/ESC/RX | 1S AIO（F4、5A ESC、ELRS 2.4G 内蔵） | 約 3.0〜3.6 g。BETAFPV F4 1S 5A AIO 級、26×26 mm マウント |
| バッテリー | 1S LiHV 300〜320 mAh、BT2.0/A30 コネクタ | 約 7.5 g。低抵抗コネクタで電圧降下を抑える |
| フレーム | 3D プリント、薄肉ガード付き | 目標 7 g 前後（市販 PP フレームは 2.7〜3.1 g、同等は狙わない） |
| 通信（操縦） | 2.4 GHz ExpressLRS、FC 内蔵受信機 | 外付け RX 分の重量を削減 |
| 通信（映像） | 必要時のみ低出力 VTX / 軽量カメラ | 標準構成では非搭載 |

### 重量バジェット（基準構成）

| 部位 | 構成 | 概算重量 |
|---|---|---|
| フレーム | 75 mm 級 3D プリント、薄肉ガード付き | 7.0 g |
| モーター | 0802 ×4 | 7.2 g |
| プロペラ | 40 mm ×4 | 1.7 g |
| FC/ESC/RX | 1S AIO、ELRS 内蔵 | 3.3 g |
| バッテリー | 1S LiHV 300 mAh | 7.5 g |
| カメラ/センサ | 軽量 FPV カメラ、ToF、LED 等 | 3.0 g |
| 配線・コネクタ・ネジ・ダンパ | 最小構成 | 1.5 g |
| ペイロード余裕 | 追加センサ、保護カバー等 | 4.0 g |
| **合計** | | **35.2 g** |

100 g 上限には大きな余裕があるが、このサイズでは 5 g の増加でも飛行特性と飛行時間に明確に影響する。
重量管理目標: フレーム単体 5〜10 g / 乾燥重量 25〜32 g / バッテリー込み 32〜40 g / ペイロード込み 35〜45 g。

### 性能概算

Happymodel EX0802 19000KV + 40 mm 4 ブレードのベンチデータ（3.7 V、1 個あたり最大 29.3 g 推力 / 4.09 A）を基準にした一次見積り:

- **推力重量比**: ベンチ値 117.2 g / 35.2 g = 3.33。電圧降下・ガード損失・個体差で 25% 減じて**実用 2.5**
- **ホバリング**: 8.8 g/モーター → 約 0.9 A ×4 + 電子機器 0.3 A = **約 3.9 A**
- **飛行時間**: 300 mAh × 80% ÷ 3.9 A ≒ **3.7 分**（市販 65 mm Whoop の公称 4.5〜5 分と整合するレンジ）
- **ペイロード余裕**: 実用 T/W 2.0 維持なら全備 44 g まで → 理論余裕 8.8 g、実用推奨 3〜6 g

## Alternatives

### プロペラ保護方式

| 案 | 説明 | トレードオフ |
|---|---|---|
| A (採用) 薄肉ガード＋リブ補強 | プロペラ周囲に 0.8〜1.2 mm 厚の環状ガード | 屋内安全性と重量のバランス。3D プリントの剛性不足→接触・振動が最大リスク |
| B フルダクト（Whoop 型） | プロペラを深い環で完全に囲う | 接触耐性最強だが 3D プリントでは重量増が大きく、空力損失・共振も出やすい |
| C オープンプロップ | ガードなし | 最軽量・高効率で飛行時間に有利。屋内接触リスクが用途要求に反する。飛行時間重視の 2 号機で検討 |

### 電源系

| 案 | 説明 | トレードオフ |
|---|---|---|
| A (採用) 1S LiHV 300〜320 mAh | 軽量標準構成 | 部品最豊富・配線簡単・安全側。大電流時の電圧降下に弱い → BT2.0 で緩和 |
| B 1S 450 mAh | 容量増 | 重量 +4〜5 g。ホバー電流も増えるため飛行時間は単純比例では伸びない。実測で不足時のみ |
| C 2S + 1102/1103 モーター | 電圧を上げ電流を下げる | 推力余裕は増すが全系統が重くなり 100 mm 級の意味が薄れる。10 g 級ペイロード要求が出た場合のみ |

### モーター KV

| 案 | 説明 | トレードオフ |
|---|---|---|
| A (採用) 0802 19000〜19500 KV | 効率・滞空寄り | ペイロード 3〜6 g 想定で電流増を抑えつつ推力余裕を確保 |
| B 0802 23000 KV | レスポンス・レース寄り | 機動性は上がるが消費電流・発熱増で飛行時間短縮。高機動版の派生とする |
| C 0702 23000〜30000 KV | 65 mm 級超軽量向け | 機体は軽くなるがペイロード余裕が小さい |

### フレーム材料（3D プリント）

| 案 | 説明 | トレードオフ |
|---|---|---|
| A (採用) PA12 / ナイロン系 | 軽量・耐衝撃・薄肉に向く | FDM ではプリント条件がシビア。SLS/MJF なら均質だが外注前提 |
| B CF ナイロン | 高剛性でアームに有利 | 割れ・異方性に注意。ガード部には硬すぎる可能性 |
| C PETG | 入手性最良・粘りあり | 剛性不足になりやすく共振→ジャイロノイズの懸念 |
| D PLA | 寸法精度は出しやすい | 脆く熱に弱い。形状確認の初期試作専用 |

市販 Whoop フレーム（射出成形 PP、2.7〜3.1 g）と同等重量を 3D プリントで出すのは非現実的。
「最軽量」ではなく「プロペラに接触しない剛性」「クラッシュで折れにくい形状」を優先し、7 g 前後を初期目標とする。

## Usage Guide

### 組み立て

1. フレームを印刷し、バリ・反り・アーム対称性を確認する ⚠️ 左右アーム長・ガード真円度の寸法誤差はそのまま振動になる
2. ダミー重量（実部品相当）を載せて重心位置を確認する
3. 0802 モーター ×4 をアーム先端マウントに M1.4 ネジで固定する ⚠️ 樹脂穴の直接ねじ込みは摩耗する。繰り返し整備を考えるとインサートか貫通ボルト化の検討要
4. AIO FC を中央プレートにソフトマウント（グロメット/ダンパ）で固定する
5. モーター配線をアームに沿わせて固定し、FC にはんだ付けする ⚠️ 遊んだ配線は振動源になる
6. プロペラを装着し、ガードとのクリアランス（全周 0.8 mm 以上）を目視・回転確認する
7. バッテリーをホルダに装着し、全備重心が中心 ±2 mm 以内にあることを確認する

### 初飛行まで

1. Betaflight でモーター回転方向・プロペラ向き・フェイルセーフ・低電圧保護を設定する
2. プロペラなしでアーム・スロットル応答を確認する
3. 固定治具上で推力・電流を実測し、概算計算（ホバー 3.9 A / T/W 2.5）と突き合わせる ⚠️ 実測が概算から大きく外れたらフレーム見直しより先に電圧降下・プロペラ接触を疑う
4. 低高度ホップ試験 → PID 調整 → ペイロード追加試験の順に段階的に進める

### 整備・運用

1. 飛行のたびにプロペラの欠け・ガード接触痕を点検する。接触痕があればクリアランスか剛性を見直す
2. バッテリーは飛行後にストレージ電圧へ。LiHV の過放電（3.0 V 以下）を避ける
3. クラッシュでアームが折れた場合はフレームのみ再印刷して電装を載せ替える ⚠️ 電装の脱着容易性（はんだレス化 or コネクタ化）はフレーム設計時に考慮する

## Concerns

1. **FDM フレームの重量超過**: 市販 PP フレーム 2.7 g に対し 3D プリント目標 7 g。これ自体は許容だが、ガードを含めて 10 g を超えると全備 40 g 目標が崩れ、飛行時間が 3 分を切る見込み。`suite.budget()` で質量予算をチェックに含める
2. **フレーム剛性不足 → ジャイロノイズ**: アーム・ガードの共振がそのまま姿勢制御悪化になる。アームは薄板ではなくリブ付き三角/箱断面とし、印刷後にねじり剛性を確認する。定量目標（固有振動数など）は LLD で設定
3. **プロペラ⇔ガード接触**: クリアランス 0.8〜1.5 mm はガードの弾性変形・印刷公差・クラッシュ変形で容易に食われる。薄肉 0.8〜1.2 mm との両立が最大の構造設計課題
4. **モーターマウント穴の摩耗**: M1.4 の樹脂直ねじは数回の脱着で緩む。穴間 6.6 mm の狭ピッチでインサートが入るかは未検証
5. **1S 大電流時の電圧降下**: フルスロットル時 10 A 超で 1S は電圧が落ちやすい。BT2.0 採用と配線最短化で緩和するが、実用最大推力は概算でも 25% 減で見ている。実測での確認要
6. **重心 ±2 mm の再現性**: バッテリー装着位置のばらつきが直接重心に効く。ホルダは位置決め形状（突き当て）を持たせる
7. **ベンチデータの流用リスク**: 性能概算は Happymodel 公表のベンチ値（4 ブレード）を 3 ブレード構成に流用している。プロペラ違いで推力・電流とも 1〜2 割ずれる可能性があり、5.4 節の飛行時間はその精度でしか信頼できない
8. **技適**: ELRS 受信機内蔵 AIO の技適有無は機種依存。国内流通品から選定する必要があり、候補が絞られる可能性
9. **未実測寸法**: 各部品の採寸（AIO 基板外形、バッテリー実寸、モーター座面）は現物入手前のカタログ値。`suite.waived()` で常時可視化する
