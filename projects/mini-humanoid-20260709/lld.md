# mini-humanoid — Low-Level Design

HLD (`hld.md`) の Decisions を実装する構築文書。座標系・全寸法・質量予算・検証基準の正本。

## Narrative

机の上に立つ、全高ちょうど 300mm の白いヒューマノイド。プロポーションは ref-1（直交ビュー）に従い、頭がやや大きく（全高の 1/5）、脚は短めでどっしりしている。白い外装シェル（頭・胸・すね）の隙間から、ダークグレーのサーボブラケットと 17 個の Dynamixel XL330 が覗く。足は大きめのフラットプレートで、爪先が前に長い。手は開いた固定爪。

構造はホビーヒューマノイドの定石「サーボブラケット直列連結」。サーボ本体が構造部材を兼ね、U 字ブラケットが関節を作る。胴体は箱型フレームで、背中側に制御基板 OpenRB-150、腹側に 2S LiPo を抱える。

## 質量予算（検証の合格ライン）

密度モデル: サーボ 1.0（=18g/個）、印刷構造材 0.8（PETG 30% インフィル相当）、シェル 0.8、バッテリー 2.0、基板 1.1。

| 項目 | 見積り | 予算上限 |
|---|---|---|
| サーボ ×17 | 306g | 306g（固定） |
| 印刷フレーム・ブラケット類 | ~190g | ≤ 230g |
| シェル（頭・胸・すね） | ~65g | ≤ 90g |
| バッテリー + 基板 | 65g | 65g（固定） |
| 配線・ファスナー（非モデル化） | +50g | 50g（固定加算） |
| **合計** | **~680g** | **≤ 800g** |

## Verification（checks.forge.js に 1:1 対応させる）

**寸法・構成:**
1. 全高: アセンブリ bbox の Z = 300 ± 3
2. エンベロープ: X170 × Y120 × Z305 の外殻との intersection 体積 = 0（bbox でなく殻法）
3. 下腿・大腿の軸間: 左右とも 50 ± 0.5（サーボ軸位置間距離）
4. サーボ数 = 17
5. 接地: 両足裏の最低点 Z = 0 ± 0.1、左右コプレーナ

**物理:**
6. 総質量（モデル質量 + 固定 50g）≤ 800g — `suite.budget()`
7. CoM: |X| ≤ 2、Y ∈ [−10, +20]、両足支持多角形（X ∈ [−48.5, +48.5], Y ∈ [−29, +44]）内
8. 干渉: 主要コンポーネント間の pairwise intersection ≤ 1mm³（締結面は共有平面のみ）
9. 左右足の内側ギャップ ≥ 5mm
10. 胴体内収納: board・battery がキャビティに完全内包、壁との intersection = 0

**印刷性:**
11. 最大印刷パーツ（torso-frame 86×72×46）が 220×220×250 ベッドに収まる
12. 最小壁厚 ≥ 2.0（設計パラメータで担保、シェル t2 / 構造 t3）

**WAIVED（常時表示）:**
- サーボ・基板・バッテリー寸法はデータシート引用の未実測プレースホルダ
- 可動域チェック（肩ロール ±80° 等）は v1 ではニュートラル姿勢のみ検証

## As-built 変更記録（v1 実装時の LLD からの差分）

| 項目 | LLD 初稿 | as-built | 理由 |
|---|---|---|---|
| 肩ピッチ軸 | Z=218 | Z=222（ロール 210 / 肘 174 / 指先 115） | 肩サーボ本体と電装の干渉回避 |
| 電装配置 | 基板を背面縦置き・バッテリー腹側縦置き | バッテリー床置き平積み + 基板をその上に平置き（z≤195） | 肩サーボ本体（z≥196）とのクリアランス確保 |
| chest-shell | 独立パーツ | torso-frame が外装を兼ねる | パーツ削減。白色仕上げで ref-3 の見た目は維持 |
| 膝サーボ本体 | 下向き吊り | 後方スイング（ふくらはぎ側）+ shank 背面プレートに溶着固定 | すね前面シェルとの干渉回避 |
| フェイスパネル | 平板 t2 | 楕円体に沿うバイザー（intersection 構成） | 平板だと四隅がスカル外へ突出するため |
| ホーン | 片側ホーン + アイドラ | 両側対称ホーン円盤 | ブロックアウト簡略化（waived で明示） |
